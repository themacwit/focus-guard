# FocusGuard Privacy Policy

**Effective date:** April 10, 2026

FocusGuard stores user preferences (toggle states) locally via `chrome.storage.sync`. This data is used solely to remember your blocking settings across browser sessions and synced Chrome profiles.

- No personal data is collected.
- No data is transmitted to any server.
- No data is shared with any third party.
- No analytics or telemetry of any kind.
- No tracking.

FocusGuard operates entirely on-device. The extension requires no host permissions beyond the content script match patterns needed to inject blocking logic on Instagram and X/Twitter.

If you have questions, open an issue on the project repository.
